# MATLAB Calculator App

## Overview
This repository contains a functional Calculator application built with **MATLAB App Designer**. The project demonstrates the use of Graphical User Interface (GUI) components and callback functions to perform mathematical operations.

## Features
* **Basic Arithmetic:** Addition, Subtraction, Multiplication, and Division.
* **Clear Function:** Reset the display for new calculations.
* **Responsive UI:** Built using MATLAB's modern App Designer framework.

## How to Run
1. Open MATLAB.
2. Navigate to this folder.
3. Open the `.mlapp` file and click **Run**.
